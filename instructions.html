Configuration Steps

Follow these steps to properly setup Elderoid. These are very detailed steps. If you think you can do it by yourself, go for it.

Good luck!

1.

First, you'll need to choose your language. Click the button indicated.

2.

Use the switch to choose your language. Only 2 languages are available.

3.

When the right language is selected, click the indicated button.

4.

Elderoid's idiom is now updated. Click the indicated button to continue.

5.

This is the welcome screen. Click the indicated button to continue.

6.

Click on Netie to test his responsiveness. He will say something to you.

7.

Click the indicated button to proceed.

8.

Now you may right your name. That's the name Netie shall address you by.

9.

When you've typed your name, click the indicated button to continue.

10.

Elderoid will now show you some features. You just need to accept or reject each of them.

11.

Read the description. Accept if you want this feature.

12.

Read the description. Accept if you want this feature.

13.

Read the description. Accept if you want this feature.

14.

Read the description. Accept if you want this feature.

15.

If you've accepted the previous feature, now you're looking at the apps intalled in your device. Click the indicated button.

16.

These are the previous list's filtering settings. Select the app categories you're interested in. Click the indicated button to return to the list.

17.

You can also look at some game suggestions. Click the indicated button.

18.

These are Elderoid's game suggestions. You might like some of these games.

19.

Back to the list. You may now choose which apps you want Elderoid to make available.

20.

Click the indicated button to proceed.

21.

Read the description. Accept if you want this feature.

22.

If you've accepted the previous feature, now you can select the temperature scale Elderoid shall use. Use the switch indicated.

23.

Once you selected the right temperature scale, click the indicated button to proceed.

24.

Read the description. Accept if you want this feature.

25.

Read the description. Accept if you want this feature.

26.

Now you'll need to allow some permissions for Elderoid to work properly.

27.

Click the indicated button for each permission.

28.

Once every permission has been allowed, click the indicated button to continue.

29.

As launcher, Elderoid needs to be setup as the device's default launcher. Click the indicated button to do such.

30.

A dialog will open. Choose Elderoid as the Default Home App.

31.

Once Elderoid is selected as the default Home App, leave the dialog box.

32.

If for some reason the previous steps didn't work, click the indicated button to go to the device's settings manually.

33.

Once Elderoid has been chosen as the default launcher app, click the indicated button to proceed.

34.

You are good to go! You can start enjoying Elderoid as it is. Click the indicated button to start!

35.

This is the Home Page. Enjoy!
